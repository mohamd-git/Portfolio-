# 🖥️ Mohammed Sharafuddin — Portfolio

A modern, terminal-themed personal portfolio built with HTML, CSS, and JavaScript. Features a hacker/dev aesthetic with glitch effects, particle animations, and smooth scroll reveals.

🔗 **Live Site:** [mohamd-git.github.io/Portfolio-](https://mohamd-git.github.io/Portfolio-/)

---

## ✨ Features

- **Terminal Aesthetic** — Monospace typography (JetBrains Mono), scanline overlays, and command-line inspired UI
- **Glitch Effects** — Animated text glitch on the hero section for a cyberpunk feel
- **Particle Canvas** — Interactive particle background on the hero section
- **Custom Cursor** — Neon green dot cursor with hover trail effects
- **Scroll Animations** — Reveal-on-scroll with clip-line, fade, and slide transitions
- **3D Tilt Cards** — Project cards with perspective tilt and sheen on hover
- **Animated Border** — Rotating conic-gradient border on featured elements
- **Dark Mode** — Full dark theme with neon green (`#00ff9c`) accent
- **Responsive Design** — Mobile-first layout with hamburger menu
- **CV Download** — Direct PDF download of resume
- **Contact Form** — Integrated contact form
- **SEO Optimized** — Meta tags, Open Graph, JSON-LD structured data

## 📂 Sections

| Section | Description |
|---------|-------------|
| **Hero** | Animated intro with name, tagline, and particle canvas |
| **About** | Background, education, and personal summary |
| **Now** | Current focus and what I'm working on |
| **Skills** | Languages, frameworks, and tools with icon pills |
| **Projects** | Featured work with GitHub links and tech stacks |
| **Achievements** | Certifications and academic accomplishments |
| **Contact** | Contact form and social links |

## 🛠️ Tech Stack

- **HTML5** — Semantic markup with accessibility attributes
- **CSS3** — Custom properties, animations, glassmorphism, `@property` hacks
- **JavaScript** — Vanilla JS for interactions, IntersectionObserver, Canvas API
- **Tailwind CSS** — Utility classes via CDN for layout
- **Google Fonts** — JetBrains Mono

## 🚀 Getting Started

No build tools required — it's a static site.

```bash
# Clone the repo
git clone https://github.com/mohamd-git/Portfolio-.git

# Open in browser
start index.html
```

Or serve locally:

```bash
npx serve .
```
